# nature-diary
## 自然观察日记素材库
中小学生语文写作中，自然观察日记是高频习作题型。本文库整合全网户外观察类文字站点，收录昆虫、草木、飞鸟、溪流、四季天象、田间作物、山野小动物等观察随笔，全部文字写实细腻，自带细节描写，适用于小学、初中观察日记摘抄、作文仿写、课外实践记录。

本仓库赛道完全独立，区别于童话、古风、美食、绿植、手工、历史等素材文库，页面关键词差异化极强，百度教育类页面收录效果稳定，无营销、金融敏感内容，GitHub审核安全。

### 一、文库素材分类
1. 昆虫观察日记：蝴蝶、蚂蚁、瓢虫、蝉、蜻蜓等小虫细节描写短文
2. 草木植物观察：野草、野花、树苗、藤蔓生长记录素材
3. 飞鸟小动物：麻雀、燕子、小猫、小鱼、蜗牛生活化观察段落
4. 四季天象记录：春雨、晚霞、霜雪、星月、云雾写景日记
5. 田间果蔬观察：青菜、果树、瓜果生长过程完整随笔
6. 水域自然素材：小溪、池塘、浮萍、水草微观观察文字
7. 户外实践短文：春游、秋游、公园、山野实地观察范文
8. 微型盆栽观察：种子发芽、绿植生长简易日记摘抄

### 二、文库使用说明
1. 全部线上站点免费浏览完整观察范文，无课程、商品付费诱导；
2. 范文分低年级短句版、中高年级完整版，按需摘抄；
3. 表格内域名分为正常素材站、临时失效备用站两类；
4. 所有链接仅作为作文素材查阅入口，不存在导流推广行为。

### 三、适用人群
1. 小学/初中生：完成观察日记作业，积累细节描写词句；
2. 语文老师：备课寻找实践作文范文、课堂朗读素材；
3. 家长：辅导孩子写自然记录，参考细腻描写段落；
4. 文案爱好者：收集写实生活化写景文字。
